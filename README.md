# Push Pull Blocks Game

A C# console-based puzzle game where players solve Push & Pull block puzzles manually or let AI algorithms solve them automatically.

## Features

* 42 playable levels
* Human and AI play modes
* Multiple search algorithms:

  * DFS
  * BFS
  * UCS
  * Hill Climbing
  * A*
* Configurable push/pull movement rules
* JSON-based level loading

## Technologies

* C#
* .NET
* JSON

## How to Run

1. Open the project in Visual Studio.
2. Build and run the application.
3. Select a level and choose Human or Computer mode.
4. If Computer mode is selected, choose a search algorithm to solve the puzzle.
